### 01 · InboxHunter — AI Desktop App
**Stack:** Tauri · React · TypeScript · Python · OpenAI

An AI-powered desktop app (Windows/Mac/Linux) that automates newsletter sign-ups and reverse outreach for lead generation. Rebuilt from a cloud model to a fully local app with $0 monthly operating cost.

- 970+ active users
- 20–30% success rate across all forms; 80–100% on compatible forms
- 50%+ reduction in LLM token costs
- Multiple 6-figure freelance income outcomes for users

### 02 · Book Blog Generator Agent
**Stack:** Python · OpenAI GPT-4o · Playwright · Google Books API

Autonomous AI agent that manages an entire book review blog pipeline — searches for books, matches the author's writing style via few-shot examples, generates full posts, and auto-publishes to Substack as drafts.

- 15+ blog posts generated
- 90% time saved per post
- Reduced creation from 2–3 hours to 15 minutes of review

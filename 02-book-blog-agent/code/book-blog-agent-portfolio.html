Book Blog Generator Agent

AI-Powered Content Generation System

Project Overview

An intelligent AI agent that automatically discovers marketing and copywriting books, then generates high-quality blog posts matching the user's unique writing style. Built with OpenAI's GPT-4o and a modular tool-based architecture.

Problem Solved

Content creators and marketers need efficient ways to:

  • Maintain consistent blog publishing schedules
  • Discover relevant books in their niche
  • Generate content that matches their personal voice and style
  • Track which topics have already been covered
  • Scale content production without sacrificing quality

System Features

Intelligent Book Discovery

  • Google Books API integration for automatic book search
  • Curated book list support (books_to_review.txt)
  • Category-based discovery: copywriting, marketing psychology, persuasion, sales
  • Duplicate detection and tracking of completed books

Five-Step Generation Workflow

1. Discover 2. Analyze Style 3. Generate 4. Save 5. Track

Style Matching

  • Learns from example blog posts in the examples/ folder
  • Analyzes heading structure, tone, format, and voice
  • Generates content that feels native to your existing blog
  • Actionable takeaways and key insights in every post

Flexible Operation Modes

  • Auto-discover mode: Finds and writes about trending books
  • Manual mode: Input specific book details for targeted posts
  • Chat mode: Interactive conversational interface with the AI agent
  • List mode: Preview discovered books before generating

Technical Implementation

AI Agent Architecture

  • OpenAI GPT-4o powered conversational agent
  • Tool-based function calling for modular capabilities
  • Persistent conversation history for context awareness
  • System prompts optimized for blog writing

Six Custom Tools

  • search_books: Query Google Books API for relevant titles
  • get_curated_books: Load pre-selected book list
  • get_written_books: Check completion history
  • load_style_examples: Analyze writing style from samples
  • save_blog_post: Write Markdown files to output/
  • mark_book_complete: Track finished posts

Data Management

  • books_to_review.txt: Curated book queue with pipe-delimited format
  • books_written.txt: Completion tracking to prevent duplicates
  • output/ folder: Generated blog posts with date-prefixed filenames
  • examples/ folder: Style reference documents

Outcome

A production-ready content generation system that:

  • Reduces blog writing time from hours to minutes
  • Maintains consistent voice across all generated content
  • Automatically discovers fresh, relevant book topics
  • Scales content output without additional human effort
  • Provides full transparency with Markdown output files
Technologies Used: Python, OpenAI GPT-4o API, Google Books API, Function Calling, Markdown, CLI Interface

Impact

Demonstrates expertise in building practical AI agents with tool-based architectures, API integrations, and style-aware content generation. Showcases the ability to create autonomous systems that solve real content marketing challenges while maintaining brand voice consistency.
